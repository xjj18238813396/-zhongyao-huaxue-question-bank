# 中药化学技术题库

静态网页题库，可直接上传到 GitHub 仓库根目录并启用 GitHub Pages。

## 题目组成

- 单选题：120 道
- 名词解释：10 道（文本框作答 + 查看参考答案）
- 简答题：8 道（文本框作答 + 查看参考答案）
- 案例分析：6 道（文本框作答 + 查看参考答案）
- 合计：144 道

## 本版新增

- 名词解释、简答题、案例分析题下方均有文本框，可保存你的作答。
- 主观题提供“查看参考答案 / 收起参考答案”。
- Word 文档中的结构式图片已打包进 `assets/`，在对应题目的题干下方直接展示：
  - 简答 1：蒽醌结构示例。
  - 简答 2：二氢黄酮、黄酮醇结构示例。
  - 案例 6：莨菪碱、山莨菪碱、东莨菪碱立体结构图。

## GitHub Pages 部署

1. 解压本压缩包。
2. 将 `index.html`、`style.css`、`questions.js`、`README.md` 与整个 `assets` 文件夹上传到 GitHub 仓库根目录。
3. 打开 `Settings → Pages`。
4. 选择 `Deploy from a branch`，分支选择 `main`，目录选择 `/(root)`，保存。

作答记录保存在当前设备、当前浏览器、当前网址中；可使用“导出记录 / 导入记录”迁移备份。
